# Java Music Player (DSA Project)

## Overview
This is a console-based music player built in **Java** using a **Doubly Linked List**.  
Each song is stored as a node in the playlist, allowing navigation forward and backward between songs.

This project demonstrates how **Data Structures and Algorithms (DSA)** concepts can be used in a practical application.

---

## Features
- Add songs to playlist
- Play current song
- Move to next song
- Move to previous song
- Remove current song
- Shuffle playlist
- Search song by name
- Display playlist
- Interactive CLI menu

---

## Data Structures Used
- **Doubly Linked List** – for playlist management  
- **Traversal** – navigating through songs  
- **Linear Search** – searching for songs  
- **Random index selection** – shuffle feature  

---

## Menu

```

1. Add
2. Play
3. Remove

5. Previous

7. Search

```

## How to Run

Compile the program:

```

javac *.java

```

Run the program:

```

java Musicplayer

```
